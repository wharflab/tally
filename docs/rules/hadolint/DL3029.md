# hadolint/DL3029

Do not use --platform flag with constant value in FROM.

> **Superseded by [`buildkit/FromPlatformFlagConstDisallowed`](../buildkit/FromPlatformFlagConstDisallowed.md)**, which provides the same check with improved diagnostics.

## Description

Using a hardcoded `--platform` value in `FROM` restricts image building to a single target platform. Use `$BUILDPLATFORM` or similar variables
instead.

## Examples

### Problematic code

```dockerfile
FROM --platform=linux/amd64 alpine
```

### Correct code

```dockerfile
FROM --platform=${BUILDPLATFORM} alpine
```
