# hadolint/DL4000

MAINTAINER is deprecated.

> **Superseded by [`buildkit/MaintainerDeprecated`](../buildkit/MaintainerDeprecated.md)**, which provides the same check with auto-fix support.

## Description

The `MAINTAINER` instruction is deprecated. Use the `org.opencontainers.image.authors` label instead.

## Examples

### Problematic code

```dockerfile
MAINTAINER moby@example.com
```

### Correct code

```dockerfile
LABEL org.opencontainers.image.authors="moby@example.com"
```
