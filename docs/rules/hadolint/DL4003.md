# hadolint/DL4003

Multiple `CMD` instructions found.

> **Superseded by
> [`buildkit/MultipleInstructionsDisallowed`](../buildkit/MultipleInstructionsDisallowed.md)**,
> which covers duplicate `CMD`, `ENTRYPOINT`, and `HEALTHCHECK`
> instructions with auto-fix support.

## Description

If more than one `CMD` instruction is listed, only the last one takes effect.

## Examples

### Problematic code

```dockerfile
FROM alpine
CMD echo "first"
CMD echo "second"
```

### Correct code

```dockerfile
FROM alpine
CMD echo "second"
```
