Do not use sudo.

| Property | Value |
|----------|-------|
| Severity | Error |
| Category | Security |
| Default | Enabled |

## Description

Do not use `sudo` as it leads to unpredictable behavior. Use a tool like [gosu](https://github.com/tianon/gosu) to enforce root.

## Examples

### Problematic code

```dockerfile
FROM busybox
RUN sudo apt-get install
```

### Correct code

```dockerfile
FROM busybox
RUN apt-get install
```
