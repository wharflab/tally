# tally-cli

A fast, configurable linter for Dockerfiles and Containerfiles.

## Installation

```bash
pip install tally-cli
```

Or run it without installing:

```bash
pipx run tally-cli lint .
uvx tally-cli lint .
```

## Usage

```bash
tally lint .
tally lint --max-lines 100 Dockerfile
```
